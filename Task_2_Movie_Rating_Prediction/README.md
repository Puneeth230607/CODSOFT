# Movie Rating Prediction

## Objective
Build a machine learning model to predict movie ratings based on features such as
genre, director, actors, duration, and release year.

## Dataset
IMDb Movies India dataset.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Model Used
- Linear Regression

## Outcome
The model predicts movie ratings and helps understand factors influencing ratings.
